Teorema de Descartes

Teorema das regras de sinais e raízes das equações

Para saber de Antemão quantas raízes reais um polinomio possui, descartes criou uma regra de sinais para saber quantas raízes positivas e negativas são possiveis em um dado polinomio dependendo da troca de sinais:

Segundo os seguintes passos:

  1. Ordenar o polinomio de forma crescente.
  2. Verificar a troca de sinais para p(x) - polinomio, Com isso determinando a quantidade de raízes positivas.
  3. Verificar a troca de sinais para p(-x) - polinomio, Com isso determinando a quantidade de raízes negativas.
  4. Subtrair dois e essas são as possibilidades.

Alguns exemplos:

Sendo: $P(x) = -4x^4 + 7x^3 + 5x^2 + 3x -2$. Devemos verificar as trocas de sinais dos coeficientes. A regra que iremos usar é: para expoentes pares os sinais se mantém e para expoentes impares trocamos

Sendo: P(x) analizaremos um por um:

  • Entre $-4x^4$ e $+7x^3$ temos uma troca de sinal de - para +.
  • Entre $+7x^3$ e $+5x^2$ não temos trocas pois se mantém de + para +.
  • Entre $+5x^2$ e $+3x$ não temos trocas pois se mantém de + para +.
  • Entre $+3x$ e $-2$ temos uma troca de sinal de + para -.

Concluindo Assim que temos 2 trocas de Sinais para P(x).

Analisando P(-x) temos:

$P(x) = -4x^4 - 7x^3 + 5x^2 - 3x -2$.

Em expoentes pares o sinal se mantém e com expoentes impares o sinal troca.

  • Entre $-4x^4$ e $-7x^3$ não temos trocas pois se mantém de - para -.
  • Entre $-7x^3$ e $+5x^2$ temos trocas de - para +.
  • Entre $+5x^2$ e $-3x$ temos trocas de + para -.
  • Entre $-3x$ e $-2$ não temos trocas pois se mantém de - para -.
